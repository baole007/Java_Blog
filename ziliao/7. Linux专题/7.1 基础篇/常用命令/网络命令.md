### write
>给用户发信息，ctrl+d保存结束。

条件:用户必须在线

### wall
>给所有用户发信息（广播信息）

`wall hello`


### ping
>给远程计算机发信息包

`ping www.baidu.com`

-c 可以指定ping几次


### ifconfig
>查看当前网络信息

eth 代表真实网卡
lo  代表回还网卡（127.0.0.1）


### mail
>查看发送电子邮件

`mail root`
ctrl+d 发送

### netstat
>显示网络相关信息

-t tcp协议
-u udp协议
-l 监听
-r 路由
-n 显示ip地址和端口号

示例：
```
netstat -tlun 查看本机监听端口
netstat -an 查看本机所有网络连接
netstat -rn 查看本机路由表
```
