### Title

给定一个Excel表格中的列名称，返回其相应的列序号。


### Demo

```
例如，

    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
    
    
```

```
示例 1:

输入: "A"
输出: 1
示例 2:

输入: "AB"
输出: 28
示例 3:

输入: "ZY"
输出: 701
```

### Analyze
得到数字，其实就是把26进制的数转换为10进制的数。算法就是基本的进制转换方法，从后往前第n位的值乘上26^(n-1)。这里26进制数是1开始的，即A是1。

### Coding
```
class Solution {
    public int titleToNumber(String s) {
        int num = 0;
        int pow = 1;
        for (int i = s.length()-1; i>=0; i--){
            num += (s.charAt(i) -'A' +1) *pow;
            pow *= 26;
        }
        return num;
    }
}
```
