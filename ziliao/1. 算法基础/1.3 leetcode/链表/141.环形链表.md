### Title

给定一个链表，判断链表中是否有环。


### Coding

方法一：哈希表
```
public boolean hasCycle(ListNode head) {
    Set<ListNode> map = new HashSet<>();
    while (head != null){
        if (map.contains(head)){
            return true;
        }else {
            map.add(head);
        }
        head = head.next;
    }
return false;
}
```

方法二：双指针

```
public boolean hasCycle(ListNode head) {
    if (head == null || head.next == null) {
        return false;
    }
    ListNode slow = head;
    ListNode fast = head.next;
    while (slow != fast) {
        if (fast == null || fast.next == null) {
            return false;
        }
        slow = slow.next;
        fast = fast.next.next;
    }
    return true;
}
```
