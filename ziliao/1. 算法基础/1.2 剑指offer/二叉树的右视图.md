### Title
二叉树的右视图


### 分析

思路可以参考按行打印二叉树。

按层放到queue中， 然后取queue每层最右边的节点, 也就是每行的最后一个节点。

### Coding
```java
class Main{
    public int[] getRightNode(TreeNode node){
        // check null
        if (node == null){
            return new int[0];
        }
        // 借助queue
        Queue<TreeNode> queue = new LinkedList<>();
        List<Integer> result = new ArrayList<>();
        List<Integer> path = new ArrayList<>();
        // 定义每行的节点个数
        int curCount = 0;
        int nextCount = 1;
        queue.offer(node);
        
        while(!queue.isEmpty()){
            curCount = nextCount;
            nextCount = 0;
            for (int i = 0; i< curCount; i++){
                TreeNode temp = queue.poll();
                path.add(temp.val);
                if(temp.left != null){
                    queue.offer(temp.left);
                    nextCount++;
                }
                if(temp.right != null){
                    queue.offer(temp.right);
                    nextCount++;
                }
            }
            // 记录path的最后一个节点并重置
            result.add(path.get(path.size()-1));
            path = new ArrayList<>();
        }
        return result;
    }
}
```
